If Streaming Doesn't Work on file://

Some browsers block streaming fetch on file:// protocol. Quick fix — use any tiny server:

# Pick one (all do the same thing):
npx serve .                      # if you have Node
python3 -m http.server 5000      # if you have Python
php -S localhost:5000             # if you have PHP

Then open http://localhost:5000
